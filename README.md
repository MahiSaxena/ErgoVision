# ErgoVision: Real-Time Desk Posture Monitor

## Overview
ErgoVision is a Computer Vision project that detects poor posture during long computer sessions using a webcam.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## Features
- Real-time webcam monitoring
- Pose landmark detection
- Neck angle calculation
- Alerts when posture becomes poor

## How to Run
Install dependencies:

pip install opencv-python mediapipe numpy

Run program:

python posture_monitor.py
